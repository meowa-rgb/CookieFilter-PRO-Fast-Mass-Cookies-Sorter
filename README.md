# CookieFilter PRO — Высокоскоростной парсер и сортировщик файлов куков (Cookies)

**CookieFilter PRO** — это специализированный автономный софт для операционной системы Windows с графическим интерфейсом (GUI), созданный для лог-хантеров, реселлеров аккаунтов и специалистов по анализу больших массивов данных. Программа автоматизирует процесс выборки целевых сессий из сотен текстовых файлов куков.

---

## Что делает софт?
* **Пакетный сбор сессий (Mass Parsing):** Сканирует указанную папку, на лету открывает все файлы `.txt` с куками и находит совпадения по ключевым доменам.
* **Сортировка по категориям:** Автоматически извлекает и распределяет куки для ключевых сервисов: **Steam, Epic Games, Roblox, Discord, Google, YouTube** в отдельные структурированные файлы.
* **Поддержка форматов:** Корректно обрабатывает базы куков в форматах Netscape и JSON, игнорируя битые символы и ошибки системных кодировок.
* **Изолированный UI:** Графическая оболочка работает в отдельном системном потоке, предотвращая зависание интерфейса при анализе файлов объемом в несколько гигабайт.
